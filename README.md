# VibeLink - AI Caption & Post App

## Project Overview
VibeLink is an Android application that allows users to generate captions for images using AI, take pictures with the camera, select images from the gallery, and post them with captions to Firebase Firestore. The app displays posts on the home page, including images and generated captions.

## Features
- Generate AI-based captions from selected or captured images.
- Capture images using the camera.
- Select images from the gallery.
- Display generated captions in the app.
- Post images with captions to Firebase Firestore.
- Display all posts on the home page.
- Simple, user-friendly UI with scrollable view and buttons.

## Screens
- Home Screen: Shows all posts from Firestore.
- Gemini Activity: Generate captions for images.
- Add Post: Upload image with caption to Firestore.
- Camera & Gallery Integration.

## Technologies Used
- Android Studio
- Java (Android)
- Firebase Firestore
- Firebase Storage
- AndroidX Libraries
- OpenAI API (Gemini) for AI caption generation
